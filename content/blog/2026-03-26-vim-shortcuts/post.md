
此处收集个人平时可能用到的一些 Vim 常用快捷键以备随时查阅。


### 1. 入门级别（基础生存技能，日常编辑够用）

按键类型：移动、插入、删除、保存退出、简单搜索。

**模式切换与基本操作：**
- **Esc** 或 **Ctrl + [**：退出 Insert 模式，返回 Normal
- **i**：在光标前进入 Insert 模式（插入）
- **I**：行首插入
- **a**：光标后追加
- **A**：行尾追加
- **o**：当前行下方新开一行并插入
- **O**：当前行上方新开一行并插入
- **:w**：保存文件
- **:q**：退出（未修改时）
- **:q!**：强制退出不保存
- **:wq** 或 **ZZ**：保存并退出
- **u**：撤销（Undo）
- **Ctrl + r**：重做（Redo）
- **.**：重复上一个命令

**光标移动（Normal 模式）：**
- **h j k l**：左 下 上 右（推荐替代方向键）
- **w / b**：下一个/上一个单词开头
- **e / ge**：单词结尾
- **0 / ^**：行首（^ 为非空白字符开头）
- **$**：行尾
- **gg / G**：文件第一行 / 最后一行
- **Ctrl + f / b**：向下/向上翻页
- **Ctrl + d / u**：向下/向上半页

**编辑与删除：**
- **x**：删除当前字符
- **dd**：删除当前行
- **dw**：删除当前单词
- **yy**（或 **Y**）：复制当前行（yank）
- **p / P**：粘贴到光标后/前
- **r**：替换单个字符（不进入 Insert）
- **J**：合并当前行与下一行

**搜索与替换：**
- **/**：向前搜索（输入后按 Enter）
- **?**：向后搜索
- **n / N**：下一个/上一个匹配
- **:s/old/new**：当前行替换（加 **g** 全行替换，如 **:s/old/new/g**）
- **:%s/old/new/g**：全文替换

**其他常用：**
- **:set number**（或 **:set nu**）：显示行号
- **:set nonumber**：隐藏行号
- **:help 关键字**：打开帮助（如 **:help w**）

这些足够应对 80% 的日常编辑。记住“动词 + 名词”思维：如 **d w**（删除单词）、**y y**（复制行）。

### 2. 进阶级别（效率提升，常用文本对象与窗口）

按键类型：掌握文本对象（text objects）和多窗口/缓冲区操作，能显著提高速度。

**文本对象（非常强大，与操作符结合使用，如 d/ciw）：**
- **iw / aw**：inner word / a word（词内/词，包括空格）
- **i( / a(** 或 **i) / a)**：括号内/外
- **i{ / a{**、**i[ / a[** 等：各种括号/引号内/外
- **it / at**：HTML/XML 标签内/外
- **ip / ap**：段落内/外

示例：
- **ciw**：修改当前单词（change inner word）
- **di(**：删除括号内内容
- **yi"**：复制双引号内内容
- **va{**：可视选中大括号外内容

**Visual 模式（选中操作）：**
- **v**：字符可视模式
- **V**：行可视模式
- **Ctrl + v**：块可视模式（列编辑神器）
- 在 Visual 中可配合 **d** 删除、**y** 复制、**>** 缩进等

**窗口与缓冲区（多文件编辑）：**
- **:sp** 或 **:split**：水平分割窗口
- **:vs** 或 **:vsplit**：垂直分割
- **Ctrl + w w**：切换窗口
- **Ctrl + w q**：关闭当前窗口
- **:e 文件名**：打开/切换文件（缓冲区）
- **:bn / :bp**：下一个/上一个缓冲区
- **:ls**：列出所有缓冲区
- **:bd**：关闭当前缓冲区

**高级移动与标记：**
- **H / M / L**：屏幕顶部/中间/底部
- **{ / }**：上一个/下一个段落
- **%**：跳转匹配括号
- **m{a-z}**：设置标记（如 **ma**）
- **'a**：跳转到标记 a 的行首
- **``**：回到上一个跳转位置
- **Ctrl + o / Ctrl + i**：跳转历史前后

**搜索替换进阶：**
- **:range s/old/new/g**（range 如 **1,10** 或 **%** 全文件）
- **:g/pattern/command**：对匹配 pattern 的行执行 command（如 **:g/foo/d** 删除含 foo 的行）
- **\* / #**：搜索光标下单词（向前/向后）

**其他实用：**
- **>> / <<**：缩进/反缩进当前行
- **> / <** 在 Visual 模式下缩进选中块
- **:set hlsearch**：高亮搜索结果

### 3. 专家级别（高级技巧，宏、寄存器、快速修复等）

按键类型：这些适合重度用户，能实现复杂批量操作和自定义。

**宏（Macro，录制重复操作）：**
- **qa**：开始录制宏到寄存器 a（q 停止录制）
- **@a**：执行宏 a
- **@@**：重复上一个宏
- 可录制复杂序列，如格式化多行代码

**寄存器（Registers，高级复制粘贴）：**
- **"ayy**：复制到寄存器 a（"a 后跟 yank）
- **"ap**：从寄存器 a 粘贴
- **"+y / "*y**：复制到系统剪贴板（需 +clipboard 支持）
- **:registers**：查看所有寄存器

**快速修复列表（Quickfix）与搜索多文件：**
- **:vimgrep /pattern/ ** 或 **:grep**：在多文件中搜索
- **:copen**：打开 quickfix 列表
- **:cnext / :cprev**：跳转下一个/上一个匹配

**全局命令与范围：**
- **:g**（global）：强大批量操作
- **:norm command**：对范围行执行 Normal 命令（如 **:%norm g~~** 大小写翻转全文件）
- **:set list**：显示不可见字符（如 Tab、空格）

**插入模式高级技巧：**
- **Ctrl + r "**：插入默认寄存器内容
- **Ctrl + r =**：插入 Vim 表达式计算结果（如 **Ctrl + r =2+3**）
- **Ctrl + n / p**：自动补全（单词）

**其他专家操作：**
- **:set paste**：进入粘贴模式（避免自动缩进问题）
- **gv**：重新选中上一次 Visual 区域
- **:marks**：查看所有标记
- **:changes**：查看更改历史
- **:tabnew / :tabn / :tabp**：标签页管理
- **Ctrl + a / Ctrl + x**：递增/递减光标下数字
- **:diffthis**：进入 diff 模式比较文件

**自定义与配置提示（.vimrc）：**
- 常用设置：**set number relativenumber**（相对行号）、**set ignorecase smartcase**（智能搜索）、**syntax on** 等。
- 映射快捷键：如 **nnoremap <leader>w :w<CR>**（leader 通常为空格或 \）。
